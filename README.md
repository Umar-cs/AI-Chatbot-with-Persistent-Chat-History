# AI-Chatbot-with-Persistent-Chat-History
This project is an AI-powered conversational chatbot built using Google's Gemini Generative AI model and the Streamlit framework. It provides an intuitive web-based interface where users can interact with the chatbot in real time while automatically preserving conversation history for future reference.
The application is designed to deliver fast, context-aware responses and maintain previous chats, creating a seamless and personalized conversational experience.

**Features**
Powered by Google's Gemini Generative AI model
Interactive and responsive web interface built with Streamlit
Automatic chat history saving and retrieval
Context-aware conversations across multiple messages
Clean and user-friendly interface
Lightweight and easy to deploy
Secure API integration
Supports continuous multi-turn conversations

**Project Workflow**
1. User Interaction
Users enter prompts through the Streamlit interface.
Messages are sent to the Gemini AI model for processing.
2. AI Response Generation
The Gemini model analyzes the conversation context.
A relevant and coherent response is generated and returned to the user.
3. Chat History Management
Every user prompt and AI response is automatically stored.
Previous conversations can be accessed to maintain context and improve the user experience.
4. Real-Time Display
The conversation updates instantly within the Streamlit application.
Both current and historical messages are presented in an organized chat interface.

**Results**

The application delivers an efficient AI chat experience with persistent conversation history, allowing users to continue previous discussions without losing context. Its lightweight architecture and simple deployment make it suitable for personal assistants, educational tools, customer support prototypes, and AI-powered web applications.
